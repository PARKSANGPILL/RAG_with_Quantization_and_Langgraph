### 사용 방법
- llama 3.1 8B모델 다운받기
```
https://huggingface.co/meta-llama/Llama-3.1-8B-Instruct/tree/main
```
___

```
git clone https://github.com/ggerganov/llama.cpp.git
```
```
cd llama.cpp
```
- requirements.txt 설치
```
pip install -r requirements.txt
```
